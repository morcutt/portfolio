Overview

Our Module object is modeled after Nicole Sullivan's Module Object. The major difference is that our Module object does not have an inner child property. Its body, header and footer are direct children. The Module object doesn't add any visual treatments. Instead, it acts as a base object to be extended by other more granular objects.
